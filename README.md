# 简单的django项目，录入每天加班数据。
## 一、主要功能
- 录入每天加班数据 
- 计算每天加班数据 
- 发送本周的加班时长到邮件中。
- 个性化倒计时
## 二、用到的技术
- 后端：python+django
- 前端：bootstrap和layui，jquery，ajax等
- 数据库：mysql
